Select 选择框

使用 ng-options 实现下拉选择

也可以使用 ng-repeat 实现

ng-repeat 指令是通过数组来循环 HTML 代码来创建下拉列表，但 ng-options 指令更适合创建下拉列表，它有以下优势： 使用 ng-options 的选项是一个对象， ng-repeat 是一个字符串。

使用 ng-options 实现 (对象): {{selectedSite}}

使用 ng-repeat 实现 (串): {{ selectedSite2 }}

数据源为对象

使用对象作为数据源, x 为键(key), y 为值(value)

你的选择为: {{ selectedSite3 }}

说明，对象作为数据源的时候选择的数据为 value , value 可以设置为对象，当然也可以直接使用某个属性值
